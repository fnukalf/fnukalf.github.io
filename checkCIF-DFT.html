Data Import

Crystal structure data is loaded by importing CIF files. To open a CIF file, select File | Open CIF from the menu bar. An Open File Dialog appears. Choose the files you wish to load and click Open. The Open CIF feature supports opening multiple files at once.

Image 1

It is necessary that a CIF file contains a structure name, complete description of the cell, space group and at least one atom, otherwise it wont be loaded.

After a structure is successfully loaded, a corresponding project is created in the Projects tree view.

Image 2

A structure loaded using the Open CIF feature is loaded as the original structure. A project containing an original structure allows for an optimized/output structure to be loaded. To do so, select File | Add Optimized Structure from the menu bar. An Open File Dialog appears. Choose the file you wish to load as the optimized structure and click Open. After an optimized structure is successfully loaded, a tree node representing the structure is created in the Projects tree view under the corresponding project tree node.

Image 3

An alternative way to add an optimized structure is the following: Place the cursor over the project tree node of the project, where you wish to load an optimized structure. Right-click on the project tree node. A context menu appears. Select Add Optimized Structure. An Open File Dialog appears. Choose the file you wish to load as the optimized structure and click Open.

Image 4

It is possible to load both an original structure and a corresponding optimized structure at the same time. To do so, it is necessary that the files are named using the following pattern: example.cif (for original structure), example-out.cif (for optimized structure). It is also necessary that both of the files are located in the same folder.

To load a pair of original and optimized structures, select File | Open CIF + Add Optimized from the menu bar. An Open File Dialog appears. Choose the files you wish to load and click Open. The program will find all the possible pairs of files named using the aforementioned pattern and will load them into new projects in the Projects tree view. The Open CIF + Add Optimized feature is useful for loading large datasets.

Atoms Filter

When loading a structure, it is possible to apply a filter for atoms.

To enter the Atoms Filter setup, select Setup | Setup checkCIF-DFT from the menu bar. The Setup checkCIF-DFT Dialog is then opened. In the Setup checkCIF-DFT Dialog, select the Import Filter tab.

Image

A correct CIF file should comply with the following standards:

       A disordered atoms label should end with the ? character, for example H1?

       An atom that is the equivalent of another atom but is translated by an integer translation should be labeled using the pattern: H1 (for the original atom), H1* (for the translated atom)

       An atom that is obtained by applying a symmetry operation to another atom should be labeled using the pattern: H1 (for the original atom), H1A (for the symmetry-generated atom). The addition to the symmetry-generated atoms label may be an arbitrary string of letters

Note that this standard may not be always satisfied by the authors of the original CIF file.

There are 2 mechanisms for filtering atoms: filtering by labels and filtering by calculation.

Filter atoms by labels

The label filter feature allows the user to filter out atoms that are labelled using the standard described above. It is possible to filter out:

       Disordered atoms

       Atoms that are equivalent to other atoms, but translated by an integer translation

       Atoms that are obtained by applying a symmetry operation to another atom

For the filter to take effect, check the checkboxes corresponding to the desired filter and click the OK button.

Filter atoms by symmetry

The symmetry filter feature allows the user to filter out atoms based on calculation. It is possible to filter out:

       Atoms that are equivalent to other atoms, but translated by an integer translation

       Atoms that are obtained by applying a symmetry operation to another atom

The symmetry filter feature is particularly useful when loading a structure, where the atom labeling standard described above is not satisfied.

For the filter to take effect, check the checkboxes corresponding to the desired filter and click the OK button.

Visualization

When a structure is loaded, checkCIF-DFT provides the visualization of the 3D model of the structure. The structure can be zoomed in/out by rolling the mouse wheel.

Image 1

The 3D model can be rotated by pressing the left mouse button and moving the mouse.

Movement

Furthermore, it is possible to move the structure within the visualization panel. One way to move the 3D model is using the blabla buttons from the menu bar. The same behavior can be achieved using the arrow keys. Note that for the arrow keys approach to work, the visualization panel has to be focused (done by left-clicking into the area of the visualization panel). Both of the approaches support press-and-hold behavior. This means that when an arrow key is pressed and held, the 3D model moves until the arrow key is released. If you want to move the 3D model back to the center, you can either click the blabla button from the menu bar or press the Home key.

Display options

The 3D visualization offers extensive displaying options. There are two ways to open the display settings:

Select View | Display options from the menu bar.

Right-click inside the 3D visualization panel. A context menu appears. Select Display Options from the context menu.

Once done, the Display Options dialog is opened. The Display Options dialog contains the following tabs:

General Tab

Lattice Tab

Picking mode and measuring

       None this mode allows for no picking, so when an atom is clicked, it is not selected

       Pick Atoms this mode allows to measure the distance between 2 atoms. In order to do so, select the Measure Distances mode and left-click 2 atoms. The 3D visualization highlights the line between the 2 chosen atoms and the distance between them (in Angstrom).

       Measure Angles This mode allows to measure the angle between 3 atoms. In order to do so, select the Measure Angles mode and left-click 3 atoms. The 3D visualization highlights the angle between the 3 chosen atoms and shows the numerical value.

       Measure Torsions This mode allows to measure the torsion between 4 atoms. In order to do so, select the Measure Torsions mode and left-click 4 atoms. The 3D visualization highlights the torsion between the 4 chosen atoms and shows the numerical value.

Measurement clean-up

Once measurements are no longer needed, it is possible to clean them up using the blabla button from the menu. All measurements are then cleaned-up at once.

Radius view restriction

The radius view restriction feature allows to restrict the view only to those atoms, that are located within a selected radius around a chosen atom. The restricted view takes into account all atoms including atoms generated by symmetry operations and periodic translations.

To activate the restricted view, click the blabla button from the menu, then select Show Atoms In Vicinity from the drop down list. This opens the Show Atoms In Vicinity Dialog.

Structure Editing

When an input structure is loaded, it is possible to edit the structure. Structure editing is done using the Edit Structure dialog. There are two ways to enter the Edit Structure dialog:

1.     In the Projects tree view, click on the tree node representing the structure you wish to edit. The chosen structure gets focused. Select Edit | Edit Structure from the menu bar. The Edit Structure dialog is then opened.

2.     In the Projects tree view, place the mouse cursor on the tree node representing the structure you wish to edit and right click the tree node. A context menu appears. Select Edit Structure from the context menu. The Edit Structure dialog is then opened.

Image 1

Note: For all changes to take effect, the OK button or Apply button need to be pressed. If, by accident, you make unwanted changes, click the Cancel button or close the Edit Structure dialog.

The Edit Structure dialog contains a table of all the atoms present in the structure. Inside the table, you may change the element type of an atom. To do so, select the row corresponding to the atom you wish to edit, unwrap the drop down list in the Symbol column, in the selection, click on the element type you wish to apply and press enter. Inside the table, you may also change the label of a selected atom and the atomic coordinates manually.

Remove Atoms

To remove atoms, select the rows corresponding to the atoms you wish to remove and click the Remove Selected button.

Add Atoms

Add Atom Feature

You may add an arbitrary atom using the Add Atom feature. Inside the Edit Structure dialog, click the Add Atom button. The Add Atom dialog is then opened. Inside the Add Atom dialog, you may specify the properties (element type, label, atomic coordinates) of the atom you wish to add. To add an atom, specify the new atoms properties and click the Add button.

Add Hydrogen Atom Feature

A hydrogen atom may be added using the Add Hydrogen Atom feature. Inside the Edit Structure dialog, click the Add H Atom button. The Add Hydrogen Atom is then opened. The Add Hydrogen Atom feature allows you to add a hydrogen atom in between 2 other atoms. This feature is particularly useful when modelling hydrogen atoms at sites where hydrogen bonds are expected.

To use the Add H Atom feature, reference atoms X1 and X2 need to be specified. To specify the reference atoms, you may follow one of the following approaches:

1.     Manually type the reference X1 and X2 atom labels in the corresponding text boxes.

2.     Click on atoms in the 3D model visualization and the atom labels will automatically update inside the X1 and X2 atom label text boxes. To return back to selecting an already specified atom, click inside the corresponding atom label text box and the possibility to select an atom will be refreshed.

After youve specified the reference atoms, the position of the hydrogen atom needs to be specified. The position of the hydrogen atom may be specified using the fractional distance of atom X1 and atom X2, where atom X1 is the starting point. This means that a fractional distance of 0 corresponds to the position of atom X1 and a fractional distance of 1 corresponds to the position of atom X2. Another way to specify the position of the hydrogen atom is by specifying the cartesian distance

To specify the position of the hydrogen atom using the fractional distance, select the Insert fractional distance radio button and inside the text box, insert the corresponding numerical value.
